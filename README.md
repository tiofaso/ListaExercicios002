Nesse projeto de java foram solicitados 7 exercícios a seguir:
01 - printar um "Hello world!"
02 - Fazer média aritimética de 4 notas 
03 - Converter metros para centímetros
04 - Calcular o salário a partir do valor/hora e horas trabalhadas
05 - Converter temperarura em celsius para fahrenheit
06 - Mostrar qual é o número antecessor e sucessor de algum número
07 - Calcular o salário bruto, líquido e os descontos em impostos
